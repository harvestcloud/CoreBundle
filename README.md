CoreBundle

[![Build Status](https://travis-ci.org/harvestcloud/CoreBundle.svg?branch=master)](https://travis-ci.org/harvestcloud/CoreBundle)

This bundle is part of the [Harvest Cloud project][1]. It contains the core
